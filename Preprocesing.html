autor: Piotr Dubak

Cel analizy¶

  1. Badana populacja: klienci sklepw spoywczych
  2. Rodzaj badania: ankieta on-line dostpna dla wszystkich uytkownikw na terenie Polski
  3. Metoda analizy: EDA ("Exploratory Data Analysis") - analiza deskrypcyjna

Analiza deskrypcyjna to proces badania, opisywania i interpretacji danych w celu uzyskania wgldu i zrozumienia ich cech, wzorcw i zwizkw. Jest to technika czsto stosowana w dziedzinach naukowych, bada spoecznych, statystyce, lingwistyce i wielu innych dziedzinach.

Gwnym celem analizy deskrypcyjnej jest opisanie i podsumowanie danych w sposb, ktry ujawnia istotne informacje. Moe obejmowa takie elementy jak obliczanie rednich, median, odchyle standardowych, minimalnych i maksymalnych wartoci, oraz prezentowanie danych w postaci tabel, wykresw lub grafw.

Analiza deskrypcyjna umoliwia identyfikacj kluczowych cech, trendw, anomalii i relacji w danych. Moe rwnie pomc w odkrywaniu wzorcw, porwnywaniu grup lub kategorii danych oraz wyprowadzaniu wnioskw na podstawie zebranych informacji.

Gwnym celem EDA jest zapewnienie wgldu w dane jeszcze przed sformuowaniem jakichkolwiek zaoe. Pomaga identyfikowa oczywiste bdy, lepiej pojmowa wzorce wystpujce w obrbie danych, wykrywa wartoci odstajce i anomalie, a take odnajdywa interesujce relacje midzy zmiennymi.

Po przeprowadzeniu analizy EDA i uzyskaniu istotnych spostrzee wci mona wykorzysta t metod do bardziej zaawansowanej analizy danych lub modelowania, w tym na potrzeby uczenia maszynowego.

Wyniki przeprowadzonej analizy pokazay, e dla wspczesnego konsumenta decyzja o zakupie produktu nie jest motywowana wycznie chci zaspokojenia okrelonych potrzeb, ale w znacznym stopniu determinuje j potrzeba demonstracji przekona, statusu spoeczno-ekonomicznego i stylu ycia. Nowoczesny model konsumpcji niesie okrelone konsekwencje dla praktyki dziaa marketingowych.

Badanie ankietowe dotyczyo postaw i zachowa respondentw w tracie dokonywania zakupw produktw ywnociowych. W tym celu zosta skonstruowany kwestionariusz ankiety

**Cechy, takie jak wiek, pe, wyksztacenie, preferowany typ sklepu, dochody netto, preferowana marka sklepu, preferowany towar, preferowany rodzaj promocji oraz czynnik zakupowy, mog wpywa na wysoko zakupw ywnociowych. **

def outliers(df, var):
    data = df[var]
 
    def detect_outliers_iqr(data):
        q1, q3 = np.percentile(data, [25, 75])
        iqr = q3 - q1
        lower_bound = q1 - (1.5 * iqr)
        upper_bound = q3 + (1.5 * iqr)
        outliers_IQR = [i for i, x in enumerate(data) if x < lower_bound or x > upper_bound]
        return outliers_IQR

    def detect_outliers_mean_std(data):
        mean = np.mean(data)
        std = np.std(data)
        lower_bound = mean - (3 * std)
        upper_bound = mean + (3 * std)
        outliers_mean_std = [i for i, x in enumerate(data) if x < lower_bound or x > upper_bound]
        return outliers_mean_std

    def detect_outliers_zscore(data):
        threshold = 2
        z_scores = zscore(data)
        outliers_zscore = [i for i, z in enumerate(z_scores) if abs(z) > threshold]
        return outliers_zscore

    def detect_outliers_winsorizing(data):
        lower_bound, upper_bound = np.percentile(data, [5, 95])
        data = np.clip(data, lower_bound, upper_bound)
        outliers_winsorizing = [i for i, x in enumerate(data) if x < lower_bound or x > upper_bound]
        return outliers_winsorizing

    # Reshape the data to a 2D array
    data = np.array(data).reshape(-1, 1)

    outliers_IQR = detect_outliers_iqr(data)
    outliers_mean_std = detect_outliers_mean_std(data)
    outliers_zscore = detect_outliers_zscore(data)
    outliers_winsorizing = detect_outliers_winsorizing(data)

    df = pd.DataFrame(data)
    df['Odstajce_IQR'] = np.where(df.index.isin(outliers_IQR), -1, 1)
    df['Odstajce_mean_std'] = np.where(df.index.isin(outliers_mean_std), -1, 1)
    df['Odstajce_Zscore'] = np.where(df.index.isin(outliers_zscore), -1, 1)
    df['Odstajce_Winsorizing'] = np.where(df.index.isin(outliers_winsorizing), -1, 1)
   
    # Dodatkowa kolumna: Czy_Odstajce
    df['Czy_Odstajce'] = np.where((df['Odstajce_IQR'] == -1) |
                                (df['Odstajce_mean_std'] == -1) |
                                (df['Odstajce_Zscore'] == -1) |
                                (df['Odstajce_Winsorizing'] == -1) , True, False)
    df = df.loc[df['Czy_Odstajce'] == True]
    df = df.rename(columns={0: var})  # Zmiana nazwy kolumny 0 na Nazwa_Zmiennej
    
    # Dodaj kolumn "ile_ident"
    df['ile_ident'] = df[['Odstajce_IQR', 'Odstajce_mean_std', 'Odstajce_Zscore', 'Odstajce_Winsorizing']].apply(lambda row: row.value_counts().get(-1, 0), axis=1)
    
    return df
